How It Works

Up and running in minutes.

No configuration files to wrestle with. No dependencies to install. Just one command.

1

Install

Interactive setup wizard. Choose modules, auto-configure your MCP client.

npx iconnect-mcp init
2

Configure

Or add a single JSON block to Claude Code, Codex, or any MCP client manually.

3

Use

AI reads your notes, manages your calendar, controls Music, sends mail, and more.

Configuration

One block. That's it.

Add this to your MCP client configuration and every Apple app is instantly available to your AI.

mcp-config.json
{ "mcpServers": { "iconnect": { "command": "npx", "args": ["-y", "iconnect-mcp"] } } }

Installation

Start in seconds.

Install globally, or run directly with npx. No build step required.

# Install globally $ npm install -g iconnect-mcp # Or run directly $ npx iconnect-mcp

Requires Node.js 18+ and macOS. Works with Claude Code, Codex, and any MCP-compatible client.
