## What is AirMCP?

AirMCP is a [Model Context Protocol](https://modelcontextprotocol.io/) server that exposes the Apple ecosystem to any MCP-compatible AI client. It bridges LLMs like Claude, GPT, and Gemini to native macOS applications through JXA (JavaScript for Automation), a Swift bridge, and direct system APIs — so the AI reads, writes, and acts on top of your Apple data. AirMCP is the connector; the AI is the thinker.

## Quick Start

```bash
npx airmcp init
```

The interactive setup wizard will guide you through module selection and MCP client configuration. See the [Installation guide](/getting-started/installation/) for detailed instructions.
