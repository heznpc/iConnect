## What is AirMCP?

AirMCP is a [Model Context Protocol](https://modelcontextprotocol.io/) server that exposes the entire Apple ecosystem to AI assistants. It bridges LLMs like Claude, GPT, and Gemini to native macOS applications through JXA (JavaScript for Automation), a Swift bridge, and direct system APIs.

  <Card title="25 Modules" icon="puzzle">

## Quick Start

```bash
npx airmcp init
```

The interactive setup wizard will guide you through module selection and MCP client configuration. See the [Installation guide](/getting-started/installation/) for detailed instructions.
